Содержание раздела

Набор скриптов для повышения эффективности работы на компьютере.

 

Для работы скриптов требуется:

  • AutoHotKey v.1.1.03.00.

Переключатель языка

(Language Switcher)

Возможности:

  • Переключение языка Enu ↔ Rus по RCtrl и по LAlt+CapsLock.
  • Преобразование Lat ↔ Rus последних набранных символов по Pause
    (направление определяется по сравнению количества Lat/Rus символов).
  • Преобразование Rus → Lat последних набранных символов по LShift+Pause.
  • Преобразование Lat → Rus последних набранных символов по RShift+Pause.
  • Преобразование Lat ↔ Rus (уже) выделенного текста по LAlt+Pause.
  • Преобразование Lat ↔ Rus последнего слова (слева от курсора) по RAlt+Pause.
  • Преобразование Lat ↔ Rus последних набранных символов по LAlt+LShift+Pause.

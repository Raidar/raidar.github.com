Изменённые схемы для Colorer

(Changed Schemes for Colorer)

Схемы для Colorer: новые, изменённые и дополненные.

Состав:

  • Каталог lib: общие основные схемы:
    • rh_defs — регионы и схемы по умолчанию,
    • rh_base — базовые (простые) регионы, сущности и схемы,
    • rh_lang — схемы различения кириллицы и латиницы (пример),
    • rh_main — основные (сложные) регионы, сущности и схемы,
    • rh_sign — схемы выделения последовательностей символов,
    • rh_math — схемы выделения математических символов и цифр/чисел,
    • rh_dict — регионы и схемы для работы с текстовыми словарями.
  • Каталог rhs: новые и дополненные схемы:
    • rhtext — схема для обычного текста,
    • rhlua — схема для исходного текста на языке Lua,
    • rhpascal — схема для исходного текста на языке Pascal,
    • rhctxdic — схема для текстового словаря формата Context v.3.51,
    • rhrme — схема для обычного текста, оформленного как ini‑файл,
    • config\rhini — схема для ini‑файла с раскраской обычного текста,
    • config\rhlum — схема для файла с меню на языке Lua (Rh_Scripts),
    • config\rhcfglua — схема для конфигурационного файла на языке Lua.
  • Каталог rhz: несущественно изменённые схемы.
  • Файлы rhs.*.ent.hrc: новые прототипы.
  • Файл rhs-proto.hrc — основной изменённый файл с прототипами.

Установка в каталог Colorer:

  1. Сделать резервную копию файла catalog.xml и папок hrc и hrd.
  2. Распаковать в каталог Colorer.
  3. Для получения исходной версии попробуйте восстановить только catalog.xml.
  4. Если нет какой‑либо цветовой схемы, проверьте содержимое hrd\catalog-console.xml.
    Если в нём нет нужной схемы, восстановите весь файл или только схему из резервной копии файла.

Отмена установки схем:

  1. Удалите файл catalog.xml и каталоги hrc и hrd.
  2. Восстановите резервную копию файла catalog.xml и папок hrc и hrd.

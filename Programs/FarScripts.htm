LuaFAR

Библиотека LuaFAR позволяет расширить возможности FAR Manager при помощи написания плагинов, скриптов и утилит на языке Lua.

Скрипты обычно реализуют простые действия, утилиты — более сложные, комплексные. Скрипты и утилиты обычно объединяются в наборы и пакеты в соответствии с предпочтениями их разработчиков.

 

Библиотека LuaFAR, LuaFAR for Editor, LuaFAR Search и ряд других плагинов разработаны Сэмуэлом Зейгерманом (Shmuel Zeigerman).

Плагины

LuaFAR for Editor (LuaFAR для Редактора) — плагин с набором утилит, производящих различные действия по управлению работой в редакторе FAR Manager.

LuaFAR Search (LuaFAR Поиск) — плагин для поиска и замены текста в редакторе FAR Manager.

Пакеты

Пакет LF context

LuaFAR context — пакет для определения типов файлов и работы с файлами конфигураций параметров, связанных с типами файлов.

Пакет позволяет задать единую конфигурацию типов файлов в разных скриптах и плагинах LuaFAR, а также использовать в них конфигурации, зависящие от типа текущего файла.

Пакет разработан на основе конфигураций, используемых в fl_scripts, доработан и расширен совместными усилиями с maxfl . LF context требуется для работы пакета Rh_Scripts, пакета LuaFAR area config, и набора fl_scripts.

 

Для работы тестовой сборки требуется:

  • Lua v.5.1.4 (lua5.1.dll или lua51.dll).
  • FAR Manager v.3.0 build 3147 .
  • LuaFAR v.3.1.1 .

Тестовая сборка v.3.0.5.2 [Build от 2013-02-06] (0,086 MiB).

Внимание: Перед установкой обязательно прочтите install.rus. Для "LuaFAR для Редактора" пакет достаточно установить в каталог плагина. Пакет можно установить в каталог, доступный для всех скриптов LuaFAR. Порядок задания такого каталога описан в файле справки context\faq\faq_rus.hlf, тема «Где размещать общие скрипты и библиотеки?».

Пакет Rh_Scripts

Rh_Scripts — пакет скриптов и утилит для "LuaFAR для Редактора".

Состав:

  • Утилита «Пользовательское меню Lua» (LUM).
  • Утилита «Прямоугольное меню» (RectMenu).
  • Общие скрипты:
    • Скрипт «Настраиваемое меню» (CustomMenu).
    • Скрипт «Фильтрационное меню» (FilterMenu).
    • Скрипт «Вызыватель меню выбранного вида» (MenuCaller).
    • Скрипт «Формирователь текста для пунктов меню» (MenuTexter).
    • Скрипт «Меню из пунктов вывода символов» (CharsItems).
    • Скрипт «Конфигуратор пакета Rh_Scripts» (rhsConfig).
  • Скрипты для редактора:
    • Скрипт «Текстовые шаблоны» (TextTemplate).
    • Скрипт «Завершение слов» (WordComplete).
    • Скрипт «Усекатель пустоты» (VoidTruncate).
    • Скрипт для использования скриптов в авторежиме.
  • Служебные скрипты для работы скриптов и утилит.

Утилита «Пользовательское меню Lua» позволяет выводить меню в зависимости от типа файла.

Возможности:

  • Привязка файла к типу с помощью пакета LuaFAR context.
  • Привязка типа к выводимому меню в lua-файле.
  • Поддерживаемые меню: прямоугольное, фильтрационное и поисковое.
  • Виды пунктов: подменю, макросы, шаблоны, скрипты и команды.
  • Назначение комбинаций клавиш на пункты меню разными способами.
  • Поддержка настроек собственных меню LUM и локализации текста.

Примеры меню LUM:

  • LuaEUM — для редактора.
  • LuaPUM — для панелей.
  • LumSVN — для TortoiseSVN.
  • LumFLS — для работы с fl_scripts.

Примеры общих подменю:

  • Разные скрипты (Other scripts):
    Общая информация, Применение LF context, Пример Hello, world!.
  • Примеры команд (Command samples):
    Показ File_id.diz различными способами.

Примеры подменю для редактора (LuaEUM):

  • Вставка шаблона (Template Insert).
    Готовые подменю для Lua, C/C++, Pascal, Html.
  • Наборы символов (Character kits):
    • Буквы (греческие, кириллица).
    • Математика (основные и другие индексы).
    • Пунктуация (символы-знаки, пробелы и черты).
  • Кавычки/скобки (Quotes/brackets):
    • Закавычивание (Quote text).
    • Раскавычивание (Dequote text).
  • Lua-скрипты (Lua scripts):
    • Чистка текста (Clear text).
    • Усечение пустоты (Truncate void).

Примеры подменю для панелей (LuaPUM).

  • Меню Git (Git menu).

Пример подменю для TortoiseSVN (LumSVN):

  • Меню команд TortoiseSVN. (TortoiseSVN menu).

Подменю LumSVN «TortoiseSVN» позволяет выполнять почти все команды TortoiseSVN.

Особенности:

  • Аналог контекстного меню TortoiseSVN Проводника.
  • Работает только с текущим элементом активной неплагиновой панели.
  • Различаются: файл/папка, рабочая копия или нет, версионированный или нет.
  • Для работы используются TortoiseProc.exe и SubWCRev.exe из TortoiseSVN.
  • Настройки хранятся в файле
    %FarProfile%\scripts\Rh_Scripts\LumSVN\config\TortoiseSVN.cfg.

Иллюстрация подменю «TortoiseSVN»:

  • версионированные папка и файл
  • папка и файл нерабочей копии,
  • неверсионированные папка и файл.

 

Утилита «Прямоугольное меню» позволяет выводить меню в несколько строк и столбцов.

Возможности:

  • Меню в виде таблицы из нескольких строк и столбцов.
  • Параметры настройки для меню в целом и для отдельных пунктов.
  • Маркирование цветом заданных символов текста пунктов меню.
  • Поддерживаемые меню: фильтрационное и поисковое.
  • Внешний вид: список, главное меню, всплывающее меню и т.д.

 

Для работы тестовой сборки требуется:

  • Lua v.5.1.4 (lua5.1.dll или lua51.dll).
  • FAR Manager v.3.0 build 3147 .
  • LuaFAR v.3.1.1.
  • LuaFAR for Editor v.3.1.2 .
  • LF context v.3.0.5.1 .

Тестовая сборка v.3.0.5.2 [Build от 2013-02-06] (0,180 MiB).

Пакет LFA config

LuaFAR area config — пакет для плагина "LuaFAR для Редактора", предназначенный для автонастройки редактора FAR в зависимости от типа файла.

Возможности:

  • Задание настроек редактора в зависимости от типа файла.
  • Показ номеров строк (слева от поля редактирования).
  • Подсветка позиции конца строки и конца файла.
  • Подсветка правой границы и последней строки файла.

Иллюстрация работы пакета: пример 1, пример 2.

 

Для работы тестовой сборки требуется:
